# Sitio estático HostYourself

Archivos clave:

- `index.html`: landing.
- `styles.css`: diseño.
- `app.js`: configurador y lógica de afiliados.
- `data/products.json`: catálogo editable de recomendaciones.
- `data/affiliate-links.json`: configuración de afiliados.

## Cambiar enlaces que caducan

No edites `index.html` para cambiar productos. Edita solamente:

```txt
data/products.json
data/affiliate-links.json
```
